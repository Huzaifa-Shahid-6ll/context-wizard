## Product Hunt Launch Copy

**Name:** Context Wizard

**Description:**

As developers, we're all excited about the potential of AI coding assistants. But we've also all experienced the frustration of getting bad suggestions because the AI doesn't understand our codebase.

Context Wizard solves this problem by automatically generating a `.cursorrules` file from any GitHub repository. This file gives your AI coding assistant the context it needs to provide you with accurate, relevant, and helpful code suggestions.

With Context Wizard, you can:

*   Generate context files for any public or private GitHub repository.
*   Download a `.cursorrules` file that works with Cursor, Windsurf, and other popular AI coding tools.
*   Save time and frustration by giving your AI the context it needs to be a true assistant.
