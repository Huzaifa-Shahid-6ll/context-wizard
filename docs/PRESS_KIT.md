## Product Description

### 50 Words

Context Wizard is a tool for developers that automatically generates context files for AI coding assistants. It analyzes any GitHub repository to create perfect context, helping you write better code faster with your favorite AI tools. Stop fighting bad context and start shipping faster.

### 100 Words

Context Wizard is a developer tool designed to solve the problem of providing accurate context to AI coding assistants. By analyzing any GitHub repository, it automatically generates `.cursorrules` and other context files that help AI tools like Cursor and Windsurf understand your codebase. This leads to more accurate code suggestions and faster development cycles. With support for private repositories and popular frameworks, Context Wizard is the easiest way to stop fighting with your AI and start building better software, faster.

### 200 Words

Context Wizard is a powerful new tool for developers that streamlines the process of working with AI coding assistants. One of the biggest challenges developers face with AI coding tools is providing them with the right context to generate accurate and relevant code. Context Wizard solves this problem by analyzing any GitHub repository—public or private—and automatically generating a `.cursorrules` file. This file provides the AI with a deep understanding of the project's structure, dependencies, and conventions.

With Context Wizard, developers can stop wasting time manually crafting context or dealing with inaccurate AI suggestions. The tool is framework-aware, meaning it can identify and prioritize important files and directories for popular technologies. This results in more relevant and helpful AI-generated code, which in turn accelerates the development process. Context Wizard supports popular AI coding tools like Cursor and Windsurf, making it an essential addition to any modern developer's workflow. Try it for free and experience the power of perfect context.

## Features

*   **Unlimited Generations:** Create as many context files as you need.
*   **Works with Your Favorite Tools:** Generate context files for Cursor, Windsurf, and other AI coding tools.
*   **Private Repo Support:** Securely connect your private GitHub repositories.
*   **Framework-Aware:** Understands popular frameworks and libraries to create more accurate context.
*   **Easy to Use:** Simple, intuitive interface to get you started in seconds.
